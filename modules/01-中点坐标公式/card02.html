例题 1 — 已知两点求中点

题目：已知 \(A(3,\ 5)\)，\(B(-1,\ 1)\)，求线段 AB 的中点 C 的坐标。

解：

设中点 \(C(x_0, y_0)\)，由中点坐标公式：

\[ x_0 = \frac{x_1 + x_2}{2} = \frac{3 + (-1)}{2} = \frac{2}{2} = 1 \] \[ y_0 = \frac{y_1 + y_2}{2} = \frac{5 + 1}{2} = \frac{6}{2} = 3 \]

答：中点 C 的坐标为 \((1,\ 3)\)。

💡 验证：A(3,5) → C(1,3) → B(-1,1)，x 坐标每次减 2，y 坐标每次减 2，对称成立。
