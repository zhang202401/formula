考试形式：闭卷笔试 90分钟 满分 100分 不可使用计算器 本专题约 10~15分
1 中点坐标公式 掌握
选择题 填空题

① 给出两点求中点坐标

例：已知 \(A(2,\ -1)\)，\(B(4,\ 3)\)，求线段 \(AB\) 的中点 \(C\) 的坐标。
解：由中点坐标公式：
\(\displaystyle x_0 = \frac{x_1 + x_2}{2} = \frac{2 + 4}{2} = 3\)，
\(\displaystyle y_0 = \frac{y_1 + y_2}{2} = \frac{-1 + 3}{2} = 1\)，
答：中点坐标为 \((3,\ 1)\)

② 已知中点与一端点求另一端点

例：已知线段 \(AB\) 的中点 \(C(1,\ 3)\)，端点 \(A(-2,\ 1)\)，求端点 \(B\) 的坐标。
解：设 \(B(x,\ y)\)，由中点坐标公式：
\(\displaystyle \frac{-2 + x}{2} = 1 \Rightarrow -2 + x = 2 \Rightarrow x = 4\)，
\(\displaystyle \frac{1 + y}{2} = 3 \Rightarrow 1 + y = 6 \Rightarrow y = 5\)，
答：端点 B 的坐标为 \((4,\ 5)\)
公式：\(\displaystyle x_0 = \frac{x_1+x_2}{2},\quad y_0 = \frac{y_1+y_2}{2}\)；\(x_2 = 2x_0 - x_1\)，\(y_2 = 2y_0 - y_1\)
2 倾斜角与斜率 理解 / 掌握
选择题 填空题

① 已知倾斜角求斜率

例：直线倾斜角 \(\alpha = 60^\circ\)，求斜率 \(k\)。
解：\(k = \tan\alpha = \tan 60^\circ = \sqrt{3}\)。
若 \(\alpha = 135^\circ\)，则 \(k = \tan 135^\circ = \tan(180^\circ - 45^\circ) = -\tan 45^\circ = -1\)。
答：\(\alpha = 60^\circ\) 时 \(k = \sqrt{3}\)；\(\alpha = 135^\circ\) 时 \(k = -1\)

② 已知两点求斜率

例：直线过点 \((-1,\ 2)\) 和 \((3,\ -2)\)，求斜率 \(k\)。
解：由两点式斜率公式：
\(k = \dfrac{y_2 - y_1}{x_2 - x_1} = \dfrac{-2 - 2}{3 - (-1)} = \dfrac{-4}{4} = -1\)。
答：\(k = -1\)

③ 特殊角的斜率值

例：求下列倾斜角对应的斜率：
解：\(k = \tan\alpha\)：
\(\alpha = 0^\circ \Rightarrow k = 0\)；\(\alpha = 30^\circ \Rightarrow k = \dfrac{\sqrt{3}}{3}\)；
\(\alpha = 45^\circ \Rightarrow k = 1\)；\(\alpha = 60^\circ \Rightarrow k = \sqrt{3}\)；
\(\alpha = 90^\circ \Rightarrow\) 斜率不存在；
\(\alpha = 120^\circ \Rightarrow k = -\sqrt{3}\)；\(\alpha = 135^\circ \Rightarrow k = -1\)；\(\alpha = 150^\circ \Rightarrow k = -\dfrac{\sqrt{3}}{3}\)。
公式：\(k = \tan\alpha\ (\alpha\neq90^\circ)\)，\(\displaystyle k = \frac{y_2-y_1}{x_2-x_1}\ (x_1\neq x_2)\)
3 直线方程 掌握
选择题 填空题 解答题

🧠 解题流程思维导图：三步法求直线方程

求直线方程 三步走
① 求斜率 \(k\)
\(k = \tan\alpha\)
例1：\(\alpha=60^\circ \Rightarrow k =\) \(\sqrt{3}\)
例2：\(\alpha=135^\circ \Rightarrow k =\) \(-1\)
\(k = \dfrac{y_2-y_1}{x_2-x_1}\)
例1：过\((0,0),(2,4) \Rightarrow k =\) \(2\)
例2：过\((-1,2),(3,-2) \Rightarrow k =\) \(-1\)
平行 ⇒ \(k_1 = k_2\)
例1：\(l_1 \parallel l_2,\ k_1=2 \Rightarrow k_2 =\) \(2\)
例2：\(l_1 \parallel l_2,\ k_1=-3 \Rightarrow k_2 =\) \(-3\)
垂直 ⇒ \(k_1 \cdot k_2 = -1\)
例1：\(l_1 \perp l_2,\ k_1=2 \Rightarrow k_2 =\) \(-\dfrac{1}{2}\)
例2：\(l_1 \perp l_2,\ k_1=-\dfrac{3}{4} \Rightarrow k_2 =\) \(\dfrac{4}{3}\)
② 看有无截距 \(b\)
有 \(b\) → 斜截式
\(y = kx + b\)
例1：\(k=2,\; b=3 \Rightarrow\) \(y = 2x + 3\)
例2：\(k=-1,\; b=4 \Rightarrow\) \(y = -x + 4\)
没 \(b\) → 点斜式
\(y-y_0 = k(x-x_0)\)
例1：过\((1,2), k=3 \Rightarrow\) \(y-2=3(x-1)\)
例2：过\((0,-1), k=2 \Rightarrow\) \(y+1=2x\)
③ 化为一般式 \(Ax + By + C = 0\)
斜截式 → 一般式
移项即可
例1：\(y=2x+3 \Rightarrow\) \(2x-y+3=0\)
例2：\(y=-x+4 \Rightarrow\) \(x+y-4=0\)
点斜式 → 一般式
展开→移项
例1：\(y-2=3(x-1) \Rightarrow\) \(3x-y-1=0\)
例2：\(y+1=2x \Rightarrow\) \(2x-y-1=0\)
公式：\(k = -\dfrac{A}{B}\)，\(b = -\dfrac{C}{B}\) \((B\neq0)\)；点斜式 \(y-y_0 = k(x-x_0)\)；斜截式 \(y = kx + b\)
4 两直线位置关系 理解 / 掌握
选择题 填空题 解答题

① 判定两直线平行/垂直

例：判断直线 \(l_1: x + 2y + 1 = 0\) 与 \(l_2: 2x + 4y - 3 = 0\) 的位置关系。
解：化为斜截式：\(l_1: y = -\dfrac{1}{2}x - \dfrac{1}{2}\)，\(l_2: y = -\dfrac{1}{2}x + \dfrac{3}{4}\)。
\(k_1 = k_2 = -\dfrac{1}{2}\) 且 \(b_1 = -\dfrac{1}{2} \neq b_2 = \dfrac{3}{4}\)，
答：\(l_1 \parallel l_2\)（平行）

② 已知平行/垂直关系求参数（用公式 \(k = -\dfrac{A}{B}\) 求斜率判断）

例1（平行）：直线 \(2x + 3y + 1 = 0\) 与 \(4x + 6y + k = 0\) 平行，求 \(k\) 的取值范围。
解：由 \(k = -\dfrac{A}{B}\) 求两直线斜率：
\(k_1 = -\dfrac{2}{3}\)，\(k_2 = -\dfrac{4}{6} = -\dfrac{2}{3}\)。
\(\because k_1 = k_2\)，两直线可能平行或重合。
将 \(l_2\) 化为斜截式：\(6y = -4x - k \Rightarrow y = -\dfrac{2}{3}x - \dfrac{k}{6}\)，\(b_2 = -\dfrac{k}{6}\)。
平行条件 \(b_1 \neq b_2\)：\(-\dfrac{1}{3} \neq -\dfrac{k}{6}\)，即 \(k \neq 2\)。
答：\(k \neq 2\) 时两直线平行
例2（垂直）：直线 \(2x + 3y + 1 = 0\) 与 \(3x - 2y + k = 0\) 垂直，求 \(k\) 的值。
解：由 \(k = -\dfrac{A}{B}\) 求斜率：
\(k_1 = -\dfrac{2}{3}\)，\(k_2 = -\dfrac{3}{-2} = \dfrac{3}{2}\)。
验证垂直条件：\(k_1 \cdot k_2 = -\dfrac{2}{3} \times \dfrac{3}{2} = -1\) ✅
\(\because k_1 \cdot k_2 = -1\) 恒成立，
答：\(k\) 为任意实数时两直线均垂直

③ 求两条直线的交点坐标

例：求直线 \(x + y - 3 = 0\) 与 \(x - y + 1 = 0\) 的交点坐标。
解：联立方程组：\(\begin{cases} x+y-3=0 \\ x-y+1=0 \end{cases}\)。
两式相加：\(2x - 2 = 0 \Rightarrow x = 1\)，
代入第一式：\(1 + y - 3 = 0 \Rightarrow y = 2\)。
答：交点坐标为 \((1,\ 2)\)
公式：平行 \(\iff k_1=k_2\) 且 \(b_1\neq b_2\)；垂直 \(\iff k_1\cdot k_2 = -1\)
5 距离公式 掌握 / 了解
选择题 填空题 解答题

① 求两点间距离

例：已知 \(A(1,\ 2)\)，\(B(4,\ 6)\)，求 \(|AB|\)。
解：由两点间距离公式：
\(|AB| = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}\)
\(= \sqrt{(4 - 1)^2 + (6 - 2)^2} = \sqrt{3^2 + 4^2} = \sqrt{9 + 16} = \sqrt{25} = 5\)。
答：\(|AB| = 5\)

② 求点到直线的距离

例：求点 \(P(1,\ -2)\) 到直线 \(3x + 4y - 5 = 0\) 的距离 \(d\)。
解：由点到直线距离公式：
\(d = \dfrac{|Ax_0 + By_0 + C|}{\sqrt{A^2 + B^2}}\)
\(= \dfrac{|3\times 1 + 4\times (-2) - 5|}{\sqrt{3^2 + 4^2}}\)
\(= \dfrac{|3 - 8 - 5|}{5} = \dfrac{|-10|}{5} = 2\)。
答：\(d = 2\)
公式：\(|AB| = \sqrt{(x_2-x_1)^2+(y_2-y_1)^2}\)；\(\displaystyle d = \frac{|Ax_0+By_0+C|}{\sqrt{A^2+B^2}}\)
6 圆的方程 了解 / 掌握
选择题 填空题 解答题

① 已知圆心和半径写标准方程

例：圆心 \(C(3,\ -2)\)，半径 \(r = 4\)，求圆的标准方程。
解：圆的标准方程为 \((x - a)^2 + (y - b)^2 = r^2\)。
代入 \(a = 3,\ b = -2,\ r = 4\)：
\((x - 3)^2 + (y + 2)^2 = 16\)。
答：\((x-3)^2 + (y+2)^2 = 16\)

② 一般式化为标准式（配方法）

例：求圆 \(x^2 + y^2 - 4x + 6y - 3 = 0\) 的圆心坐标和半径。
解：配方：
\(x^2 - 4x = (x - 2)^2 - 4\)，
\(y^2 + 6y = (y + 3)^2 - 9\)，
代入得 \((x - 2)^2 + (y + 3)^2 - 4 - 9 - 3 = 0\)，
\((x - 2)^2 + (y + 3)^2 = 16\)，
\(\therefore\) 圆心 \((2,\ -3)\)，半径 \(r = \sqrt{16} = 4\)。
答：圆心 \((2,\ -3)\)，半径 \(r = 4\)

③ 待定系数法求圆的方程

例：求以 \(A(1,\ 2)\)、\(B(3,\ 4)\) 为直径端点的圆的方程。
解：圆心为 \(AB\) 的中点：
\(a = \dfrac{1+3}{2} = 2\)，\(b = \dfrac{2+4}{2} = 3\)，即圆心 \((2, 3)\)。
半径 \(r = \dfrac{|AB|}{2} = \dfrac{\sqrt{(3-1)^2 + (4-2)^2}}{2} = \dfrac{\sqrt{4+4}}{2} = \dfrac{2\sqrt{2}}{2} = \sqrt{2}\)。
代入标准方程：\((x - 2)^2 + (y - 3)^2 = 2\)。
答：\((x-2)^2 + (y-3)^2 = 2\)

④ 判断二元二次方程是否表示圆

例：判断 \(x^2 + y^2 + 2x - 4y + 5 = 0\) 是否表示圆。
解：圆的一般式表示圆的条件：\(D^2 + E^2 - 4F > 0\)。
这里 \(D = 2,\ E = -4,\ F = 5\)，
\(D^2 + E^2 - 4F = 2^2 + (-4)^2 - 4\times5 = 4 + 16 - 20 = 0\)。
由于 \(D^2 + E^2 - 4F = 0\)，方程不表示圆（表示一个点）。
答：不表示圆

⑤ 已知圆上三点求圆的方程（待定系数法）

例：圆过三点 (A(0,0))，(B(2,0))，(C(0,2))，求圆的方程。
解：设圆的一般式 (x^2 + y^2 + Dx + Ey + F = 0)，代入三点：
代入 (A(0,0))：(F = 0)。 代入 (B(2,0))：(4 + 2D = 0 Rightarrow D = -2)。 代入 (C(0,2))：(4 + 2E = 0 Rightarrow E = -2)。
得 (x^2 + y^2 - 2x - 2y = 0)，配方得 ((x-1)^2 + (y-1)^2 = 2)。
答：(x^2 + y^2 - 2x - 2y = 0) 或 ((x-1)^2 + (y-1)^2 = 2)
公式：圆心 \(\left(-\dfrac{D}{2},\ -\dfrac{E}{2}\right)\)，半径 \(r = \dfrac{1}{2}\sqrt{D^2+E^2-4F}\)
7 直线与圆的位置关系 理解
选择题 填空题 解答题（压轴）

① 判断直线与圆的位置关系（\(d\) 与 \(r\) 比较）

例：判断直线 \(2x + y + 5 = 0\) 与圆 \(x^2 + y^2 - 10x = 0\) 的位置关系。
解：将圆化为标准式：\(x^2 - 10x + y^2 = 0 \Rightarrow (x - 5)^2 + y^2 = 25\)，
圆心 \(C(5, 0)\)，半径 \(r = 5\)。
圆心到直线的距离：\(d = \dfrac{|2\times5 + 0 + 5|}{\sqrt{2^2 + 1^2}} = \dfrac{15}{\sqrt{5}} = 3\sqrt{5} \approx 6.708\)。
\(\because d > r\)（\(6.708 > 5\)），
答：直线与圆相离

② 由位置关系求参数

例：直线 \(3x + 4y + k = 0\) 与圆 \((x-3)^2 + y^2 = 4\) 相切，求 \(k\)。
解：圆心 \(C(3, 0)\)，半径 \(r = 2\)。
相切条件：\(d = r\)，即 \(\dfrac{|3\times3 + 4\times0 + k|}{\sqrt{3^2 + 4^2}} = \dfrac{|9 + k|}{5} = 2\)。
\(|9 + k| = 10\)，\(\therefore 9 + k = 10\) 或 \(9 + k = -10\)，
\(k = 1\) 或 \(k = -19\)。
答：\(k = 1\) 或 \(k = -19\)

③ 求弦长（垂径定理）

例：求圆 \((x-2)^2 + (y+3)^2 = 4\) 被直线 \(x - y - 3 = 0\) 截得的弦长。
解：圆心 \(C(2, -3)\)，半径 \(r = 2\)。
圆心到直线距离：\(d = \dfrac{|2 - (-3) - 3|}{\sqrt{1^2 + (-1)^2}} = \dfrac{|2 + 3 - 3|}{\sqrt{2}} = \dfrac{2}{\sqrt{2}} = \sqrt{2}\)。
由垂径定理，弦长 \(= 2\sqrt{r^2 - d^2} = 2\sqrt{4 - 2} = 2\sqrt{2}\)。
答：弦长为 \(2\sqrt{2}\)
公式：弦长 \(= 2\sqrt{r^2 - d^2}\)（垂径定理）
8 圆的切线 理解 / 初步掌握
解答题（压轴）

① 过圆上一点求切线

例：过点 \(P(1,\ -4)\) 且与圆 \((x+1)^2 + (y+3)^2 = 5\) 相切的直线方程。
解：第一步：验证点 \(P\) 是否在圆上。
代入：\((1+1)^2 + (-4+3)^2 = 4 + 1 = 5\)，\(\therefore P\) 在圆上 ✅
第二步：求圆心 \(C(-1, -3)\)，则 \(k_{CP} = \dfrac{-3 - (-4)}{-1 - 1} = \dfrac{1}{-2} = -\dfrac{1}{2}\)。
第三步：由切线 ⊥ 半径，得 \(k_l = -\dfrac{1}{k_{CP}} = 2\)。
第四步：点斜式 \(y - (-4) = 2(x - 1)\)，即 \(y + 4 = 2x - 2\)，
整理得 \(2x - y - 6 = 0\)。
答：\(2x - y - 6 = 0\)

② 过圆外一点求切线

例：过点 \(P(0,\ 5)\) 向圆 \(x^2 + y^2 = 9\) 作切线，求切线方程。
解：第一步：验证 \(P\) 到圆心距离 \(= 5 > 3\)，\(P\) 在圆外，应有两条切线。
第二步：设切线斜率为 \(k\)，方程为 \(y = kx + 5\)，即 \(kx - y + 5 = 0\)。
第三步：由圆心 \((0,0)\) 到切线距离等于半径 \(r = 3\)：
\(d = \dfrac{|5|}{\sqrt{k^2 + 1}} = 3 \Rightarrow 5 = 3\sqrt{k^2 + 1} \Rightarrow k^2 + 1 = \dfrac{25}{9} \Rightarrow k^2 = \dfrac{16}{9} \Rightarrow k = \pm \dfrac{4}{3}\)。
第四步：代入得两条切线：\(y = \dfrac{4}{3}x + 5\) 和 \(y = -\dfrac{4}{3}x + 5\)。
化为一般式：\(4x - 3y + 15 = 0\) 和 \(4x + 3y - 15 = 0\)。
第五步：检查斜率不存在的情况：\(x = 0\) 到圆心距离 \(0 < 3\)，不是切线。
答：两条切线方程为 \(4x - 3y + 15 = 0\) 和 \(4x + 3y - 15 = 0\)
核心：切点与圆心连线 ⊥ 切线；圆心到切线距离 \(d = r\)
